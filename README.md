# 2048_like

## Synchronisation des codes

```sh
npx cap sync
```

## Ouvrir Android Studio

```sh
npx cap open android
```

## Vérifier si le téléphone est connecté

```sh
adb devices
```

## Lancer le débogage sur téléphone

```sh
npx cap run android
```

Pour lancer le server
npx vite

Pour faire le build
npx vite build
